About this app

  • Instructions for using the app
      • Press the w key to move up
      Press the s key to move down
      Press the d key to move to the right and if you reach the end of the row it will continue to the next row
      Press the a key to move to the left and if you reach the end of the row it will continue to the previous row
      Press the x key to select the item highlighted
  • Mini-Test Conditions
      • Comparing two different versions of my interface: 4-key vs. 5-key. Both conditions will be timed to see which is more efficient
      The 4 key switch control will not include the use of the 'a' key so the user is only allowed to utilize the w, s, d, and x keys
      The 5 key switch control will include the use of the all the keys so the user is allowed to utilize the w, s, d, a, and x keys
  • Example Problems
    1. 78 / 4
    2. 95 * 3
    3. 61 + 30
    4. 83 - 45
    5. 27 * 52
    6. 45 - 54
