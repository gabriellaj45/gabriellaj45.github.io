Projects

Game Changer

Many board games present their information visually, such as text on the cards and the location of the game pieces. This causes accessibility challeneges for people with visual impairments. Game Changer is a tool to be utilized by people with visual impairments to receive information about the state of a board game. Players are able to get the information on cards read and the location of their game pieces as well as their opponent's.

LabelGen

People with visual impairments label cards with Braille to make them accessible. However, this process can be ardous. LabelGen provides a quicker and more affordable option to do-it-yourself accessibility methods or purchasing accessibility kits from sites like 64oz Games and MaxiAids. LabelGen uses a template of the important regions of the card established through an interdependent relationship between blind and sighted players. It uses these templates when it scans the card to extract the information, to translate it into Braille, and print out a Braille label. If the card contains too much text, then a QR code is printed to be scanned by the mobile application that will relay all the textual information in an audio format.
