Directions for Using Calculator

This calculator uses a different type of notation then today's modern calculators. To use this calculator, you must first enter the operation you wish to perform, for example the addition sign, followed by a space then the first number you wish to calculate followed by a space then the second number and then select the equals button or press the enter key. This calculator also works with the keyboard butyou must first navigate using the tab key or select the text field with your mouse or by touch. The table belows shows the correct text representations of the operations.
An example of the correct input would be: + 4 5
