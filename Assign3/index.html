Help
This tab provides a table of all the keyboard shortcuts that this app contains and all the accessibility features of the app.
  • Tab Control
    • Every button in the app is accessible via the tab key
  • Keyboard Shortcuts
    • Below is the list of keyboard shortcuts that will allow you to navigate through the app quickly and selects key on the dialer.
  • Contact List Table
    • The contact list table was changed from a list of buttons to a table so that the screen reader can have an easier time traversing the information.
  • Captioned Images
    • All images in this app have alt text meaning that there is a description attached to all images explaining what the images is about and the objects contained in it. This allows a screen reader to provide its user with information about the picture.
  • ARIA Landmark Roles
    • The ARIA Landmark Roles provides the screen with directions of how to navigate through the app.
  • ARIA Live Regions
    • The ARIA Live Regions alerts a screen reader user to any content changes.
